Local RAG Evaluation Harness

A retrieval system built to say "I don't know" rather than fabricate, with a reference-based evaluation harness measuring retrieval accuracy, abstention correctness, and answer faithfulness across a labelled question set.

Runs fully locally on Ollama by default — this hosted demo uses Groq for public availability.
